# Screenshots - Como Adicionar

## Instruções

1. Salve seus prints de tela nesta pasta com nomes descritivos:
   - `01-home.png` - Tela inicial/dashboard
   - `02-login.png` - Página de login
   - `03-calcular.png` - Calculadora de tributos
   - `04-contratos.png` - Lista de contratos
   - `05-novo-contrato.png` - Formulário de novo contrato
   - `06-detalhes-contrato.png` - Detalhes de um contrato
   - `07-admin.png` - Área administrativa (rota privada)

2. Os formatos recomendados são: PNG, JPG ou GIF

3. Mantenha o tamanho das imagens razoável (< 1MB cada)

4. Após adicionar as imagens, faça commit:
   ```bash
   git add docs/screenshots/
   git commit -m "docs: adiciona screenshots da aplicação"
   git push origin main
   ```
